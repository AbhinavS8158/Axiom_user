
🏢 Axiom User App

Axiom User App is a real estate property browsing and booking application developed to simplify the process of searching, exploring, and booking properties.

This module is designed specifically for users who are looking for properties in different categories, including:

🏠 Rental Properties
🏢 Properties for Sale
🛏 PG (Paying Guest) Accommodation

The application allows users to search properties, view complete details, save favorites, and proceed with bookings in a structured and user-friendly way.

🚀 Key Features

🔐 User Authentication (Login / Signup)
🔎 Advanced Property Search
🏘 Property Categories (Rent / Sale / PG)
📄 Detailed Property View
❤️ Save / Favorite Properties
📅 Booking Management
💬 Real-time Chat with Service Providers
🌐 API Integration for Property Data

🛠 Tech Stack

Framework: Flutter
Language: Dart
State Management: Provider
Backend: Firebase
Database: Cloud Firestore
Media Storage: Cloudinary

📂 Project Structure
lib/
│
├── models/        → Property & user data models  
├── services/      → API & backend services  
├── controllers/   → Business logic  
├── screens/       → UI screens  
├── widgets/       → Reusable UI components  
└── main.dart      → Application entry point  

⚙️ Installation Guide
Clone the repository:
git clone https://github.com/yourusername/axiom-user-app.git

Navigate into the project:
cd axiom-user-app

Install dependencies:
flutter pub get

Run the app:
flutter run
